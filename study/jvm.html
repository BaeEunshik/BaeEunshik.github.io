JVM이란

JVM - Java Virtual Machine (자바 가상 머신)

자바 바이트코드는 JRE(Java Runtime Environment)위에서 동작한다.
JRE에서 가장 중요한 요소가 JVM(Java Virtual Machine)이며
JRE는 자바 API와 JVM 으로 구성되어있다.

JVM의 역할은 자바 애플리케이션을 클래스 로더(Class Loader)를 통해 읽어들여
자바 API와 함께 실행하는 것이다.

JVM(Java Virtual Machine) 에서의 가상머신(Virtual Machine)은
프로그램을 실행하기 위해 컴퓨터와 유사한 머신을 소프트웨어로 구현한 것이라 할 수 있다.
자바는 본래 WORA(Write Once Run Anywhere)를 컨셉으로 설계되었다.
동일한 자바 실행 코드를 수정 없이 여러 하드웨어에서 동작시키도록 하는 역할을
JVM이 맡고 있는 것이다.

JVM의 특징

  • 스택 기반의 가상 머신
    - JVM은 스택기반으로 동작한다.
    - 인텔의 x86이나 ARM 아키텍처는 하드웨어가 레지스터 기반으로 동작한다.
  • 심볼릭 레퍼런스
